```

또는

## 디자인 생성

### 새 화면 만들기
```
@stitch 피트니스 앱 홈 대시보드 만들어줘. 다크모드에 코랄 오렌지 포인트 컬러로.
```

### 상세 프롬프트로 만들기
```
@stitch 모바일 로그인 화면 만들어줘.
- 다크 테마
- 이메일/비밀번호 입력 필드
- 소셜 로그인 버튼 (Google, Apple)
- "비밀번호 찾기" 링크
- 둥근 모서리 버튼
```

### 특정 스타일로 만들기
```
@stitch 미니멀한 일본풍 차 쇼핑몰 상품 상세 페이지 만들어줘. 
중립적인 색상, 검은색 버튼, 우아한 세리프 폰트.
```

### 웹/모바일 지정
```
@stitch 웹용 랜딩 페이지 만들어줘. SaaS 서비스 소개용.
```

```
@stitch 모바일용 설정 화면 만들어줘.
```

## 디자인 DNA 추출

### 기존 화면에서 디자인 시스템 추출
```
@stitch 방금 만든 홈 화면에서 디자인 컨텍스트 추출해줘
```

### DESIGN.md 파일로 저장
```
@stitch 피트니스 대시보드에서 디자인 컨텍스트 추출해서 DESIGN.md 작성해줘
```

### 추출되는 정보
- 색상 팔레트 (Primary, Secondary, Background 등)
- 폰트 정보 (Font Family, Weight, Size)
- 레이아웃 구조
- 컴포넌트 스타일 (버튼, 카드, 입력 필드 등)
- 아이콘 스타일

@stitch 방금 만든 화면 코드 가져와줘

```
@stitch 운동 기록 화면 코드 추출해서 workout-detail.html로 저장해줘

## Designer Flow (핵심 워크플로우)

> MCP의 진짜 가치: **일관된 디자인 시스템 유지**

### Step 1: 첫 화면 생성
```
@stitch 피트니스 앱 홈 대시보드 만들어줘. 
다크모드, 코랄 오렌지(#fb7051) 포인트 컬러, Lexend 폰트.
```

### Step 2: 디자인 DNA 추출
```
@stitch 방금 만든 홈 화면에서 디자인 컨텍스트 추출해서 DESIGN.md 작성해줘
```

### Step 3: 같은 스타일로 추가 화면 생성
```
@stitch DESIGN.md 컨텍스트 사용해서 프로필 설정 화면 만들어줘
```

```
@stitch 이 디자인 시스템으로 운동 기록 상세 화면 만들어줘
```

### Step 4: 코드 추출 및 개발
```
@stitch 모든 화면 코드 추출해서 /components 폴더에 저장해줘
```

## 🎯 실전 예시

### 예시 1: 완전한 앱 UI 제작

```
# 1. 메인 화면 생성
@stitch 암호화폐 트래킹 앱 대시보드 만들어줘. 
다크 테마, 네온 그린 포인트, 모바일용.

# 2. 디자인 시스템 추출
@stitch 디자인 컨텍스트 추출해서 DESIGN.md 저장해줘

# 3. 추가 화면들 생성
@stitch 이 디자인으로 코인 상세 화면 만들어줘
@stitch 이 디자인으로 포트폴리오 화면 만들어줘
@stitch 이 디자인으로 설정 화면 만들어줘

# 4. 코드 추출
@stitch 모든 화면 코드 가져와줘
```

### 예시 2: 기존 디자인 기반 확장

```
# 1. 기존 프로젝트 확인
@stitch 내 프로젝트 목록 보여줘

# 2. 특정 프로젝트 화면 확인
@stitch "Fitness App" 프로젝트 화면 목록 보여줘

# 3. 기존 화면에서 디자인 추출
@stitch 홈 대시보드에서 디자인 컨텍스트 추출해줘

# 4. 새 화면 추가
@stitch 이 디자인으로 커뮤니티 피드 화면 만들어줘
```

### 예시 3: 빠른 프로토타이핑

```
# 한 줄로 바로 생성
@stitch SaaS 랜딩 페이지 만들어줘. 히어로 섹션, 기능 소개, 가격표, CTA 버튼 포함. 모던하고 깔끔한 스타일.
```

## ⚡ 팁

1. **구체적으로 요청하기**: 색상, 폰트, 레이아웃 등 상세히 명시하면 결과가 좋음

2. **한 번에 하나씩**: 수정할 때는 한 가지만 요청 (여러 개 섞으면 예측 불가)

3. **Designer Flow 활용**: 첫 화면 → DNA 추출 → 추가 화면 순서로 일관성 유지

4. **DESIGN.md 활용**: 디자인 시스템 문서화해두면 나중에 재사용 가능

5. **코드 추출 후 개발**: Stitch 코드는 HTML/Tailwind CSS 기반, Next.js나 React로 변환 가능
